# Порядок запуска

## Полный запуск

```powershell
cd E:\projects\Project_Vibe\openclaw_mako_youtube
.\start.bat
```

## Ручной запуск

```powershell
cd E:\projects\Project_Vibe\openclaw_mako_youtube
docker compose up -d db

$env:DATABASE_URL = "postgresql+asyncpg://openclaw:openclaw@localhost:5433/openclaw_mako"
$env:PYTHONIOENCODING = "utf-8"
python -m uvicorn main:app --host 127.0.0.1 --port 8000
```

Во втором окне `PowerShell`:

```powershell
cd E:\projects\Project_Vibe\openclaw_mako_youtube
$env:API_BASE_URL = "http://127.0.0.1:8000"
$env:PYTHONIOENCODING = "utf-8"
python ui_launcher.py
```

## Проверка API

```powershell
Invoke-WebRequest -UseBasicParsing http://127.0.0.1:8000/health
```

## Остановка

```powershell
docker compose down
```

## Порядок действий в UI

1. Нажать `Fetch Chart` и дождаться завершения обработки.
2. После заполнения очереди нажать `Ctrl+A`.
3. Запустить bulk `★ Fill+Approve` с настройками по умолчанию.
4. Нажать `Export DOCX -> Export all filtered to DOCX`.
